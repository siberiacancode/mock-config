### Overview

`mcs run` â€” Start a mock server. It starts a mock server in the current directory using a configuration file (`.mock-config`).

### Using

```bash
npx mcs run [options]
```

### Arguments

The `run` command accepts optional CLI arguments to override config values:

- `baseUrl` (string): override the base URL path from config. Must start with `/`.
- `port` (number): override the server port from config.
- `staticPath` (string): override the static assets path from config. Must start with `/`.
