---
title: Getting Started
description: Install and set up mock-config-server
---

Install the core package using your preferred package manager:

```package-install
mock-config-server
```

Create a `mock-server.config.ts` file with server configuration and export it as default. This is the default file path, but you can [change it](/docs/cli/run).

<Tabs items={['REST', 'GraphQL']}>
  <Tab value='Rest'>

```typescript
export default [
  {
    baseUrl: '/api'
  },
  {
    configs: [
      {
        path: '/user',
        method: 'get',
        routes: [{ data: { emoji: 'ðŸ§Š', name: 'Siberia can code' } }]
      }
    ]
  }
] satisfies MockServerConfig;
```

  </Tab>

<Tab value='GraphQL'>

```typescript
export default [
  {
    baseUrl: '/graphql'
  },
  {
    configs: [
      {
        operationType: 'query',
        operationName: 'GetUser',
        routes: [{ data: { emoji: 'ðŸ§Š', name: 'Siberia can code' } }]
      }
    ]
  }
] satisfies MockServerConfig;
```

  </Tab>
</Tabs>

Start the mock server:

```bash
npx mock-config-server
```

> If the package is already installed, you can use short command `mcs`
